TabMaster

更快地管理和切换你的标签页

一个基于 WXT 框架与 React 的现代化浏览器扩展，支持 Chrome/Edge/Firefox，采用 Manifest V3 标准。 提供快速搜索、智能分组、一键切换等功能，让您的标签页管理更加高效便捷。

🚀 立即安装

🚀 核心特性

⚡

快速搜索

实时搜索所有窗口中的标签页，支持标题、URL和域名搜索，快速定位目标标签。

🗂️

智能分组

按域名自动分组显示标签，让相关标签聚集在一起，提高浏览效率。

🎯

一键切换

快速激活或关闭标签页，支持批量操作，让标签管理变得简单高效。

🔒

隐私保护

支持隐私模式，所有数据完全在本地处理，不会上传到任何服务器。

🌐

多浏览器支持

完美支持 Chrome、Edge、Firefox 等主流浏览器，一次开发，处处可用。

⚙️

现代化架构

基于 Manifest V3 标准，使用 TypeScript 和 React 开发，代码质量有保障。

🛠️ 安装使用

Chrome/Edge 安装步骤

1

安装扩展

从 Chrome 应用商店搜索并安装 TabMaster 扩展

2

开始使用

安装完成后，点击浏览器工具栏的 TabMaster 图标即可开始使用

🔐 权限说明

本扩展需要以下权限来提供完整功能：

  • tabs: 读取和管理标签页信息
  • windows: 管理浏览器窗口
  • storage: 保存用户设置和偏好
  • activeTab: 与当前活动标签页交互
  • sessions: 管理浏览会话
  • <all_urls>: 显示所有网站的标签页信息

🔒 隐私承诺

所有数据完全在本地处理，不会上传到任何服务器。我们承诺保护您的隐私安全。
